# WNNCNXMhrr

hrr - High Refresh Rate, The firmware is designed in such a way that the phone can work smoothly at 120 Hz.

### Flashing guide
- Unpack archive on C: drive,
- Start flash_all.bat,
- Format data via recovery,
- Reboot,

### Rom features
- Pure CN expirience
- Multilanguage
- 0 Mods
